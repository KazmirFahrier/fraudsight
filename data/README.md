# Data

Raw data is **not committed** (see `.gitignore`). Download one of:

- **Kaggle Credit Card Fraud** — 284,807 txns, 492 frauds (~0.172%). Place `creditcard.csv` here.
  https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- **IEEE-CIS Fraud Detection** — raw features for SQL feature-engineering practice.
  https://www.kaggle.com/competitions/ieee-fraud-detection

After download you should have e.g. `data/creditcard.csv`.
